# Smartphones dataset description

This dataset provides a comprehensive collection of information about all the latest smartphones available in the market as of the current time. The dataset was created by web scraping reputable online sources to gather accurate and up-to-date information about various smartphone models, their specifications, features, and pricing.

```brand_name``` - the brand or manufacturer of the smartphone.\
```model``` - the name and model of each smartphone.\ + \n
```price``` - the current market price of the smartphone. + \n
```avg_rating``` - ratings. + \n
```5G_or_not``` - availability or unavailability of 5G network. + \n
```processor_brand``` - the manufacturer of the processor. + \n
```num_cores``` - number of cores of the processor. + \n
```processor_speed``` - the speed of the processor. + \n
```battery_capacity``` - capacity of the battery. + \n
```fast_charging_available``` - availability or unavailability of fast charging. + \n
```fast_charging``` - the maximum fast-charging power the phone supports, in watts (W). + \n
```ram_capacity``` - capacity of the ram. + \n
```internal_memory``` - internal storage capacity in GB (e.g., 64, 128, 256) + \n
```screen_size``` - display screen size in inches (e.g., 6.1, 6.5) + \n
```refresh_rate``` - screen refresh rate in Hz (e.g., 60, 90, 120) + \n
```num_rear_cameras``` - number of rear/back cameras on the device + \n
```os``` - operating system (e.g., iOS, Android) + \n
```primary_camera_rear``` - megapixels of the primary rear camera (e.g., 12, 48, 64) + \n
```primary_camera_front``` - megapixels of the front/selfie camera (e.g., 12, 16, 32) + \n
```extended_memory_available``` - binary indicator (0 or 1) showing if the phone + \n
supports expandable storage via SD card (0 = no, 1 = yes) + \n
```resolution_height``` - screen resolution height in pixels (e.g., 1792, 2532) + \n
```resolution_width``` - screen resolution width in pixels (e.g., 828, 1170) + \n


The dataset contains 980 smartphones (981 rows including the header) with
comprehensive specifications covering pricing, performance, camera capabilities,
display features, and connectivity options.
